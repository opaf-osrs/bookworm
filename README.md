# Bookworm

A RuneLite plugin that tracks your OSRS book collection and syncs it to the Bookworm web tracker.

## Features

- Detects books as you pick them up in-game
- Shows a collection log-style popup when a new book is found
- Syncs your collection to the Bookworm web app — an interactive world map showing all 200+ collectible books
- Web app shows zone unlock chains, skill requirements, and collection progress per zone
- Per-account persistence

## Web App

The companion web tracker lets you view your progress on an interactive OSRS map, see which zones you've completed, and plan which books to collect next.

## Installation

Install via the RuneLite Plugin Hub — search for **Bookworm**.
